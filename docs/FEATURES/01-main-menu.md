# 01. 메인 메뉴

출처: PDF p.10~11 (메인 메뉴)

## 개요

애플리케이션 진입 시 표시되는 최상위 메뉴. 기능별 하위 메뉴로 진입하는 선택 화면과, 전체 시료에 대한
요약 정보(대시보드)를 함께 보여준다.

## 요구사항

1. 기능별 선택 화면(Display)을 제공한다. 최소한 아래 6개 메뉴로 진입할 수 있어야 한다.
   - 시료 관리
   - 시료 주문(접수)
   - 주문 승인/거절
   - 모니터링
   - 생산 라인
   - 출고 처리
   - (+ 종료)
2. 전체 시료에 대한 요약 정보를 진입 시점에 확인할 수 있어야 한다. 최소 항목 예시(PDF p.11 예시 화면 기준):
   - 등록 시료 종수
   - 총 재고 수량
   - 전체 주문 건수
   - 생산라인 대기 건수
   - (선택) 조회 시각
3. 메뉴 화면의 구체적인 레이아웃/문구는 자유롭게 결정한다 (PDF는 예시일 뿐 강제 스펙이 아님).

## 입력/출력

- 입력: 메뉴 번호(정수)
- 출력: 하위 메뉴로 라우팅, 잘못된 입력 시 재입력 안내

## 예외 처리

- 숫자가 아닌 입력, 정의되지 않은 메뉴 번호 입력 시 오류 메시지를 출력하고 메인 메뉴에 머무른다.

## 참고 예시 UI (PDF p.11, 자유롭게 재구성 가능)

