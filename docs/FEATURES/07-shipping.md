# 07. 출고 처리

출처: PDF p.22~23 (출고 처리)

## 개요

재고가 충분해진(=출고 준비가 끝난) `CONFIRMED` 상태 주문에 대해 출고를 처리한다.

## 기능 요구사항

1. `CONFIRMED` 상태의 주문 목록(출고 가능 주문)을 표시한다.
2. 특정 주문을 선택해 출고를 실행한다.
3. 출고 실행 시 주문 상태를 `CONFIRMED` → `RELEASE`로 전환한다.
4. 처리 일시를 기록한다.

## 검증 규칙

- `CONFIRMED` 상태가 아닌 주문은 출고 대상 목록에 노출하지 않는다.
- 이미 `RELEASE` 상태인 주문에 대한 재출고 시도는 거부한다.

## 상태 전이

```
CONFIRMED --(출고 처리)--> RELEASE
```
